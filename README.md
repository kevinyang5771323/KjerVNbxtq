# 膳食健康系统简介 python584

## 项目概述

膳食健康系统是一个基于科学管理日常饮食理念构建的平台，旨在通过定制营养方案和追踪摄入数据，帮助用户平衡营养素摄入，预防饮食相关的健康问题。

## 技术栈

- **前端技术**：Vue2.0、Element-ui
- **后端技术**：Python3.7、Flask

## 功能模块

- 用户端：浏览食谱、食物成分、博客分享等内容
- 管理端：管理食谱、食物、博客、问答等信息，提供增删改查及分类管理功能

## 系统角色

- 用户：浏览、搜索和获取定制营养方案
- 管理员：管理后台内容，包括食谱、食物信息、博客和问答等

## 运行环境

- **开发工具**：PyCharm + VSCODE
- **数据库**：MySQL5.7（及以上）
- **数据库管理工具**：Navicat10+
- **Python**：Python3.7
- **前端Nodejs**：14
- **浏览器**：谷歌浏览器
